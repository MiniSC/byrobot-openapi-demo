## 百应开放API Java Demo

### 一、简介
百应API开放，详细接口文档，请参考：http://api.byrobot.cn/doc/v1/

此代码为java调用demo，仅供与参考，调用方尽量融合自己代码中是http连接池中。

### 二、使用流程
客户先联系百应同学开通isv账号(appkey)，并建立测试企业。这样isv就可以随意调用测试企业的任务。

### 三、sample流程


sample

1、获取可管理的企业列表

2、根据企业id获取该企业的可用的机器人

3、创建任务

4、向任务中导入客户

5、开始任务

6、获取未开始的客户列表

7、获取已经完成客户列表

8、获取通话详情

其他api：

1、获取任务列表

2、获取任务详情

3、暂停和终止任务

4、获取公司的主叫号码
